# Tokyo Memories 📸🗼

A web application built to cherish and share the memories of my trip to Tokyo from **April 29 to May 9, 2026**. This project is dedicated to the wonderful friends I made during my journey, allowing us to easily look back at our time together.

## The Story Behind the App

While exploring Tokyo, I bought a tiny, toy-like souvenir camera called the **Pieni M**. Naturally, I took it everywhere with me and captured the trip through its lens.

However, the Pieni M is only equipped with a **1.22-megapixel sensor**. The resulting photos are beautifully nostalgic but undeniably low in resolution and quality. Viewing them individually on a large, modern screen didn't quite capture the magic.

To solve this, I created **Tokyo Memories**. This web application allows you to take these low-resolution photos and arrange them into beautiful, customizable collage frames—like vintage film strips or Polaroid grids. By stitching multiple small photos together, their lo-fi, retro aesthetic shines, transforming blurry snapshots into atmospheric, high-quality collages that you can download and share.

## Features

- **Interactive Canvas Editor**: Drag, drop, scale, and rotate photos on a freeform canvas.
- **Multiple Frame Layouts**: Choose from classic 4-cut film strips, square, portrait, or landscape Polaroid-style frames.
- **High-Quality Export**: Export your beautifully arranged lo-fi collages into high-resolution images ready for social media or printing.
- **Mobile-Friendly**: Fully responsive design so you can create collages on the go.

## Tech Stack

- **React 18** (Vite)
- **TypeScript**
- **Zustand** (State Management)
- **CSS Modules** (Responsive styling)

## Getting Started

First, install the dependencies:

```bash
bun install
```

Then, start the development server:

```bash
bun run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the app.
